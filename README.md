# The Birdie Club v7

A browser-based pixel golf idle / management prototype.

## What's new in v7

- Added **Flamingo** as a fully playable Birdie with new sprite assets
- **Bird unlock progression** by Club Level
- **Course unlock progression** by Club Level
- **Working shop gear**:
  - Big Beak Driver
  - Soft Feel Balls
  - Lucky Feather Tee
- **Daily tournament** mode with bonus rewards
- Better shot feedback with quality labels like Perfect / Great / Scrappy / Wild
- Improved hazard handling and Lucky Tee protection for one water ball per round
- Updated save key: `birdieClubSaveV7`
